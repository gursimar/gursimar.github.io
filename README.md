# Gursimran's webpage

## Instructions

### Local deployment
- `bundle install`
- `bundle exec jekyll serve`

### Remote source
- `git pull origin source`
- `git add .`
- `git commit -m "blah blah"`
- `git push origin source`
  
### Remote master deployment
- `bundle exec jekyll serve (locally)`
- `rake deploy`

### Rejected! master
- Go to branches on github
- Delete master
- 'rake deploy'
